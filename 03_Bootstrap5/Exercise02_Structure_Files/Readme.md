# Bootstrap Structure and Files

## CSS Folder

Contains Bootstrap styling files.

Example:
bootstrap.min.css

Used for:
- Layout
- Typography
- Buttons
- Forms
- Utilities

## JS Folder

Contains Bootstrap JavaScript plugins.

Example:
bootstrap.bundle.min.js

Used for:
- Modals
- Accordions
- Collapse
- Dropdowns
- Tooltips

## Icons Folder

Contains Bootstrap Icons.

Examples:
- bi-house
- bi-person
- bi-search

Used to display scalable vector icons.

## bootstrap.bundle.min.js

Includes:
- Bootstrap JavaScript
- Popper.js

Required for interactive Bootstrap components.
