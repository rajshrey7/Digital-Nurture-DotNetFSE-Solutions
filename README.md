# Digital Nurture 4.0 - .NET Full Stack Engineer Solutions

## Overview

This repository contains solutions for the Cognizant Digital Nurture 4.0 .NET Full Stack Engineer (FSE) learning program.

The repository is organized module-wise and contains practical exercises covering:

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* jQuery
* ANSI SQL Using MySQL
* C#
* ADO.NET

## How to Run

### HTML / CSS / JavaScript

Open the project folder and launch:

```bash
index.html
```

or use Live Server in VS Code.

### SQL

Execute:

```sql
schema.sql
sample_data.sql
```

before running exercise queries.

### C#

Navigate to the exercise folder and run:

```bash
dotnet run
```
